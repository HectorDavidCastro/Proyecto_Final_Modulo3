### Proyecto Final del Modulo 3

Una aplicacion simple de gestion de recetas de cocina hecha con React y Vite

---
### Tecnologias utilizadas
- React
- Vite
- JavaScript
- CSS
- LIbrerias como : react-hook-form, react-router-dom o zustand
- Guardado en GitHub
- Desplegado en Vercel

---
### Funcionalidades

- Presenta un listado de recetas las cuales se pueden guardar en la misma pagina, mediante las rutas de react-router y el manejo del estado global de zustnad.

- Detalles de Recetas que se obtienen mediante el consumo de una API externa, esto logrado mediante un custom hook.

- Presenta una pagina simple de LogIn aplicando otro custom hook ademas de las herramientas de react-hook-form.
